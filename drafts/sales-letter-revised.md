## 区間1｜ファーストビュー

### 何度も挫折してきたあなたへ

# X運用を「頑張る作業」から
# 「続けられる仕組み」へ

虎威（TORAI）は、AIによる投稿作成と予約投稿を組み合わせ、X運用を続けやすくする自動化サービスです。

- 投稿内容をAIでまとめて作成
- 内容を確認・編集して選べる

その繰り返しを、あなた一人で抱える必要はありません。

Googleアカウントでの登録は無料です。

**［Googleアカウントで無料登録］**

登録後に限定価格をご確認いただけます。

## 区間4｜虎威とは？

# 投稿作成から予約投稿までを、一つの流れに

虎威は、Googleスプレッドシートを使って投稿を管理し、AIによる投稿作成と予約投稿を行うサービスです。

投稿データは、あなたのGoogle環境で管理できます。

### 投稿をまとめて作成

キーワードをもとにAIで複数の投稿案を作成します。

作成された投稿は、公開前に自由に確認・編集できます。

### 投稿する日時を設定

投稿ごとに日時を指定したり、投稿間隔を設定してまとめて予約したりできます。

### 設定した時間に自動投稿

投稿時間を待って、毎回手動で操作する必要はありません。

### 続ける作業を仕組みに任せる

やる気のある日にまとめて準備し、あとは設定した流れに任せられます。

あなたが毎日頑張らなくても、X運用を続けやすくなります。

## 区間6｜開始は3ステップ

# 難しそうに見えても、開始時の設定は3段階です

### STEP 1｜プログラムをコピー

## 区間7｜料金

# X運用を仕組みに変える費用

虎威の月額プランは、1,280円です。

X APIは虎威とは別に、利用量に応じた費用が発生します。

### 月500投稿を行う場合の目安

| 内容 | 月額目安 |
| --- | ---: |
| 虎威 月額プラン | 1,280円 |
| X API 500投稿分の試算 | 約720円 |
| 合計 | 約2,000円 |

> X APIの料金は為替、利用量、X側の料金変更などにより変動します。最新の料金はX Developer Platformでご確認ください。

投稿数が増えると、X APIの費用も増えます。

虎威は、投稿すれば必ず収益が発生することを保証するサービスではありません。

投稿作成と予約投稿にかかる作業を減らし、発信を続ける仕組みを提供します。

## 区間10｜よくある質問

### いつでも解約できますか？

はい。クレジットカード契約はStripeのサブスクリプション管理画面から解約できます。解約後も契約期限までは利用でき、次回更新日以降は課金されません。

### 銀行振込でも利用できますか？

はい。銀行振込は6ヶ月プランで利用できます。事務手数料と振込手数料が別途必要です。

### スマートフォンから使えますか？

ブラウザから利用できますが、画面の操作性を考慮し、設定や投稿管理はPCの利用をおすすめします。

### X APIの費用はいくらですか？

利用量に応じてX側で課金されます。虎威の利用料金とは別に発生します。

### サポートはありますか？

メール、問い合わせフォーム、アプリ内メッセージからお問い合わせいただけます。原則24時間以内に返信します。

### 自動投稿でXアカウントが停止されませんか？

アカウントの状態は、投稿頻度や内容など複数の要因に左右されます。自動投稿を利用する場合も、Xの利用規約とルールに沿って運用してください。

### AIで投稿を作成する費用はかかりますか？

虎威ではGoogle Geminiを利用します。利用条件や無料枠はGoogle側で変更される場合があるため、最新情報をご確認ください。

## 制作時の共通ルール

- CTAの文言と色を全区間で統一する
- CTAを区間1、3、6、8、11に配置する
- 「無料」はアカウント登録に対してのみ使用する
- 割引率や初月価格を掲載する場合は、実際の設定値と一致させる
- 収益、表示回数、継続率など、確認できない成果を保証しない
- 重要な説明を画像内だけにせず、HTMLテキストでも掲載する
- モバイルでは本文16px以上、CTA文字18px以上を目安にする
- 長いFAQはアコーディオン表示にする
