# 虎威 QUESTセールスレター

## 確認方法

`index.html` をブラウザで開いてください。

## ファイル構成

- `index.html`: セールスレターHTML
- `letter.css`: レスポンシブCSS
- `assets/images/`: 使用画像
- `preview-desktop.png`: PC全長プレビュー
- `preview-mobile.png`: スマートフォン全長プレビュー

## 新規生成した画像

- `hero.png`: ファーストビュー
- `problem.png`: 投稿を続けても反応がなく疲れている人物
- `education-cycle.png`: 投稿・確認・対話・分析・改善の循環
- `pay-per-use.png`: 固定料金から従量課金への変化

## 再利用した実画面

- `screen-ai.jpg`: AI投稿作成画面
- `screen-schedule.jpg`: 一括予約画面
- `screen-analytics.png`: 分析機能の開発イメージ
- `screen-referral.png`: 紹介報酬プログラム画面

CTAリンクはすべて `https://torai.try-try.com/auth/signin` です。

## 画像生成プロンプトの方針

画像生成は組み込みのImageGenを使用しました。虎威の濃紺・青・紫・金を基調に、問題提起、マーケティング循環、X API従量課金の変化を、文字やCTAを画像内へ埋め込まず表現しています。ファーストビューのみ、直前に制作した虎威の編集可能SVGから書き出した画像を再利用しています。
