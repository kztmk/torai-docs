# 利用前チェック

虎威を利用する前に、以下を準備します。

## 必要なもの

- 虎威のサブスクリプション契約
- 投稿に使用する X アカウント
- X Developer Console の利用準備
- Google アカウント
- Google シート
- Google Apps Script
- Google ドライブ
- Gemini API Key

## 費用が発生する可能性があるもの

- 虎威の月額または年額サブスクリプション
- X API の従量課金
- Google Gemini API の利用量に応じた費用

## この章で説明すること

- サブスクリプション開始前に確認すること
- X API 従量課金の考え方
- Google 側で準備するアカウントと権限
- 複数 X アカウント運用時の注意点
