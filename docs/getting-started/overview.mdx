# 利用前チェック

虎威を利用する前に、以下を準備します。

## 必要なもの

- 虎威のサブスクリプション契約（クレジットカード・銀行振り込み）
- 投稿に使用する X アカウント
- Google アカウント
- Gemini API Key（AIでポスト作成する場合）
- Discode アカウント（虎威の投稿結果を受信する場合）

## 費用が発生する可能性があるもの

- 虎威の月額または年額サブスクリプション
- X API の従量課金

## この章で説明すること

- サブスクリプション開始前に確認すること
- X API 従量課金の考え方
- Google 側で準備するアカウントと権限
- 複数 X アカウント運用時の注意点
