# Google アカウントの準備

虎威は Google シート、Google Apps Script、Google ドライブを使って投稿データや画像・動画を管理します。

![虎威自動投稿フロー](/img/getting-start/torai_flow.jpg)

## 準備するもの

- Google アカウント
- Google シートを作成できる権限
- Apps Script をデプロイできる権限
- Google ドライブへ画像・動画を保存できる容量

## 注意点

Google シートと Apps Script は、投稿処理を実行する中心になります。

投稿用の Google アカウントを分けたい場合は、X API キー、Google シート、虎威の設定がどのアカウントに紐づくかを記録してください。
