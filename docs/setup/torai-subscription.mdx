# サブスクリプションについて

虎威を安定してお届けし続けるために、虎威はサブスクリプション制です。

ハードウェアの追加、メンテナンス費用の上昇で会費は値上げする場合があります。しかし、原則として入会時の月額・年額で更新できるようにしています。

:::caution
キャンペーンで初回割引が適用されていても更新は入会時の月額・年額となります。割引価格での更新ではありません。
:::

## サブスクリプション料金

### クレジットカード

| 期間       | 料金     |
| ---------- | -------- |
| 毎月支払い | 1,280円  |
| 半年支払い | 6,600円  |
| 年支払い   | 12,000円 |

### 銀行振り込み

　半年支払い： 6,600円　＋ 事務手数料 880円　= 合計 7,480円

## 入会方法

[虎威](https://torai.try-try.com)へアクセスします。

![虎威トップページ](/img/setup/g-sheet/step10_torai_signin.jpg)

Googleアカウントでサインインしてください。

規約への同意をお願いします。

![規約への同意](/img/setup/g-sheet/step10_agree.jpg)

規約へ同意いただくとサブスクリプションページが開きます。

![サブスクリプションの選択](/img/getting-start/torai/torai-subscription-select.jpg)

### クレジットカードでの契約

クレジットカードでは、

- 毎月の支払い
- 半年毎の支払い
- 年間での支払い

を選択していただけます。

いずれも、いつでも解約できます。解約は返金はありませんが、当初の契約期間内は使用を続けることがｄけいます。次回の更新が行われません。

「クレジットカードでお申込」ボタンをクリックすると決済ページが表示されます。

![サブスクリプション選択](/img/getting-start/torai/subscription01.jpg)

Stripeを通じた決済ですのでクレジットカード情報は当方へは一切知らされません。

![Stripe決済ページ](/img/getting-start/torai/subscription02.jpg)

決済が完了しますと、サブスクリプション完了画面になります。Stripeのお客様情報ページはボタンクリックで表示します。解約も行えます。

![サブスクリプション完了](/img/getting-start/torai/subscription03.jpg)

### 銀行振り込みでの契約

銀行振り込みでは、半年毎の支払いとなります。

- 事務手数料 880円
- 振込手数料

を別途ご負担いただくようになります。ご了承ください。

「銀行振込でお申込み」ボタンをクリックします。

![銀行振り込み](/img/getting-start/torai/bankTransfer1.jpg)

事務手数料についての確認をお願いします。

![銀行振り込み](/img/getting-start/torai/bankTransfer2.jpg)

サインインに使用されたメールアドレスへ銀行振り込みに関する情報を送らせていただいています。

![銀行振り込み](/img/getting-start/torai/bankTransfer4.jpg)

銀行振り込みが完了いたしましたら、申請ページから振り込み名義をお知らせください。。

![銀行振り込み](/img/getting-start/torai/bankTransfer3.jpg)

振り込みが確認でき次第サブスクリプションが有効になります。

![銀行振り込み](/img/getting-start/torai/bankTransfer5.jpg)

振り込み確認完了のメールも送信されます。
![銀行振り込み](/img/getting-start/torai/bankTransfer6.jpg)

また、虎威にサインインいただければ、メッセージでもお知らせします。
![銀行振り込み](/img/getting-start/torai/bankTransfer7.jpg)
![銀行振り込み](/img/getting-start/torai/bankTransfer8.jpg)

## 退会方法

虎威は、いつでも退会することができます。退会しても契約期間内は続けて使用可能です。

また、「プロフィール ＞ アカウント情報」でアカウントを削除しなければ、いつでも再開できます。

### クレジットカード

「プロフィール　＞ サブスクリプション」にある「契約情報を変更・確認する」ボタンをクリックします。
![サブスクリプション情報](/img/getting-start/torai/subscription03.jpg)

Stripeのお客様個人ページが表示されます。「Cancel subscription」をクリックします。
![サブスクリプションキャンセル](/img/getting-start/torai/subscription04.jpg)

理由を尋ねられますが、「Continue to cancellation」ボタンをクリックします。
![サブスクリプションキャンセル理由](/img/getting-start/torai/subscription05.jpg)

「Cancel subscription」ボタンをクリックします。
![サブスクリプションキャンセル実行](/img/getting-start/torai/subscription06.jpg)

プランがキャンセルになったことを確認してください。次回更新時の課金は行われません。
![サブスクリプションキャンセル完了](/img/getting-start/torai/subscription07.jpg)

### 銀行振り込み

銀行振り込みでのサブスクリプションは、期限の10日前に更新のご案内をメールで行います。入金を行わなければ、期限をもって自動的に退会となります。
