# 虎威へのサインインと基本設定

虎威を使い始めるには、虎威へサインインし、Google Apps Script の Web アプリ URL を登録します。

このページでは、虎威と Google Apps Script を連携し、本人確認を完了するまでの手順を説明します。

## 1. 虎威へサインインする

虎威のログイン画面を開き、登録済みのメールアドレスまたは案内された方法でサインインします。

> キャプチャー画像：虎威のログイン画面

ログイン後、左側メニューまたはプロフィールメニューから「プロフィール」または「APIキー」画面を開きます。

> キャプチャー画像：虎威ログイン後のサイドメニューでプロフィールを開く状態

## 2. APIキー画面を開く

プロフィール内の API キー設定画面を開きます。

この画面では、次の情報を登録します。

- Google Apps Script の Web アプリ URL
- Gemini API Key
- ChatGPT API Key
- Anthropic API Key
- Discord Webhook URL

自動投稿だけを始める場合、まず必要なのは Google Apps Script の Web アプリ URL です。

> キャプチャー画像：虎威の APIキー設定画面

## 3. GAS URL 変更を開始する

APIキー画面で「GAS URL変更」または GAS URL を変更するボタンをクリックします。

> キャプチャー画像：APIキー画面で GAS URL 変更ボタンをクリックする状態

Apps Script のデプロイ完了画面でコピーした Web アプリ URL を貼り付けます。

> キャプチャー画像：GAS URL 入力欄に Web アプリ URL を貼り付けた状態

:::caution
Apps Script の URL は `/exec` で終わる Web アプリ URL を使用します。Apps Script エディタの URL や、スプレッドシートの URL ではありません。
:::

Web アプリ URL のコピー、Google Sheet の再読み込み、本人確認コードの確認までをまとめて確認したい場合は、[Apps Script の貼り付け・デプロイ・虎威連携](./apps-script-deploy-link) を参照してください。

## 4. 本人確認コードを作成する

GAS URL を入力すると、虎威から Apps Script へ本人確認コードの作成を依頼します。

Google シートを開き、本人確認コードが作成されていることを確認します。

> キャプチャー画像：Google シートに本人確認コードが表示された状態

表示された本人確認コードをコピーします。

## 5. 虎威へ本人確認コードを貼り付ける

虎威の APIキー画面へ戻り、本人確認コード入力欄へコピーしたコードを貼り付けます。

> キャプチャー画像：虎威の本人確認コード入力欄にコードを貼り付けた状態

「保存」または「確認」ボタンをクリックします。

連携に成功すると、虎威アカウントと Apps Script が紐づきます。

> キャプチャー画像：GAS本人確認が完了した状態

:::caution
この紐づけは乗っ取り防止のため、別の虎威アカウントへ移転できません。別アカウントで使う場合は、新しい Google シートと Apps Script を作り直してください。
:::

## 6. Gemini API Key を登録する

AI による投稿案作成を使う場合は、Gemini API Key を登録します。

APIキー画面の Gemini API Key 欄へキーを貼り付け、保存します。

> キャプチャー画像：Gemini API Key を入力した状態

ChatGPT や Anthropic を使う場合も、同じ画面で必要な API Key を登録します。

:::info
AI API の利用料金は各サービス側の料金体系に従います。大量生成を行う前に、利用量と課金設定を確認してください。
:::

## 7. Discord通知を使う場合

X への自動投稿が完了したときに Discord へ通知したい場合は、Discord Webhook URL を登録します。

通知をONにし、Webhook URL を入力して保存します。保存後、Webhook URL は再表示されません。

> キャプチャー画像：Discord通知設定をONにし、Webhook URL を入力した状態

「テスト送信」ボタンがある場合は、保存前または保存後にテスト送信を行い、Discordへメッセージが届くことを確認します。

> キャプチャー画像：Discord にテストメッセージが届いた状態

## 8. 連携後に確認すること

連携が完了したら、次を確認してください。

- GAS URL が保存されている
- GAS本人確認が完了している
- 使用する AI API Key が保存されている
- Discord通知を使う場合はテスト送信が成功している
- X API キー 4 種を取得済みである

次に、[X アカウント管理](../usage/x-accounts) で投稿先の X アカウントを登録します。

## よくある失敗

### 「このGASは別の虎威アカウントに連携されています」と表示される

その Apps Script は、すでに別の虎威アカウントと連携済みです。セキュリティ上、別アカウントへの移転はできません。

新しい虎威アカウントで使う場合は、必要なシートデータだけをコピーし、新しい Google シートと Apps Script を作成してください。

### 本人確認コードが見つからない

開いている Google シートが、Apps Script を作成したシートと同じか確認してください。

### 保存しても連携できない

Apps Script の Web アプリ URL が正しいか確認してください。`/exec` で終わる URL を使用します。

### Discord通知のテストが届かない

Webhook URL が正しいか、Discord側でチャンネルやWebhookが削除されていないか確認してください。
