# API 利用量や課金が不安なとき

外部 API の費用は虎威のサブスクリプションとは別に発生する場合があります。

## X API

X Developer Console で Usage と課金設定を確認します。

大量の予約投稿やスレッド投稿を実行する前に、投稿予定数と利用量を確認してください。

## Gemini API

Google 側の管理画面で利用量と課金設定を確認します。

AI 投稿案を大量に生成する前に、利用量の上限や請求設定を確認してください。
