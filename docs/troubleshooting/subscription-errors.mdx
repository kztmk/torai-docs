# サブスクリプションが有効にならない

虎威のサブスクリプションが有効にならない場合は、契約状態と支払い状態を確認します。

## 確認すること

- 支払いが完了している
- 契約対象のアカウントでサインインしている
- 契約更新日を過ぎていない
- 解約済みまたは支払い失敗になっていない
- ブラウザを再読み込みしても状態が変わらない

## サポートへ連絡する前に

契約アカウント、発生日時、表示されているエラーメッセージを控えてください。
