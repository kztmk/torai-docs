# 投稿できない

X への投稿が失敗する場合は、以下を確認します。

## 確認すること

- X API キー 4 種が正しい
- App permissions が Read and Write 以上になっている
- 権限変更後に Access Token を再生成している
- X API の課金設定や Usage に問題がない
- Google Apps Script の Web アプリ URL が正しい
- トリガーが動作中になっている
- Google シートの Errors シートに記録がある
