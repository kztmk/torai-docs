# X へのポスト作成

X に投稿するポストを虎威で作成します。

虎威では、X の投稿画面に近い操作で本文、画像、動画、投稿予定時刻を設定できます。AI API Key を登録している場合は、キーワードから投稿案を作成することもできます。

## 1. 投稿先の X アカウントを開く

左側メニューの「X アカウント」から、投稿先のアカウントを選択します。

> キャプチャー画像：X アカウント一覧から投稿先アカウントを選択する状態

アカウント詳細画面または投稿一覧画面が開きます。

> キャプチャー画像：選択した X アカウントの投稿管理画面

## 2. 新しいポストを作成する

「新規作成」または投稿作成ボタンをクリックします。

> キャプチャー画像：新規ポスト作成ボタンをクリックする状態

本文入力欄に投稿内容を入力します。

> キャプチャー画像：ポスト本文を入力した状態

文字数が多すぎる場合は、X 側で投稿できません。画面上の文字数表示や警告を確認してください。

## 3. 画像または動画を追加する

画像や動画を添付する場合は、メディア追加ボタンからファイルを選択します。

> キャプチャー画像：画像または動画を追加するボタン

虎威では Google ドライブ上のファイルを使用します。ファイルのアップロードや選択が必要な場合は、画面の案内に従って進めます。

> キャプチャー画像：Google ドライブ上の画像を選択している状態

:::caution
動画はファイルサイズが大きくなりやすく、Apps Script の実行時間制限に影響します。大きな動画を投稿する場合は、事前にテスト投稿を行ってください。
:::

## 4. 投稿予定時刻を設定する

投稿する日時を設定します。

> キャプチャー画像：投稿予定日時を入力する状態

今すぐ投稿したい場合でも、まず少量のテスト投稿で動作確認することをおすすめします。

## 5. 保存する

内容、投稿先、メディア、投稿予定時刻を確認し、「保存」をクリックします。

> キャプチャー画像：ポスト作成画面で保存ボタンをクリックする状態

保存後、投稿一覧に戻り、作成したポストが表示されていることを確認します。

> キャプチャー画像：作成したポストが投稿一覧に表示された状態

## 6. AI で投稿案を作成する

Gemini API Key などを登録している場合は、キーワードから投稿案を作成できます。

> キャプチャー画像：AI投稿案作成画面

キーワード、投稿テーマ、口調、作成数などを入力し、投稿案を生成します。

生成結果を確認し、必要に応じて修正してから保存します。

:::caution
AI が生成した文章は必ず確認してください。誤情報、過度な表現、意図しない表現が含まれる場合があります。
:::

## 投稿前のチェック

- 投稿先アカウントは正しいか
- 本文に誤字や不要なURLがないか
- 画像や動画は正しいか
- 投稿予定時刻は意図した日時か
- スレッドにする投稿ではないか
- API利用量や課金に問題がないか
