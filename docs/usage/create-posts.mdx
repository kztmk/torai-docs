# X へのポスト作成

X に投稿するポストを虎威で作成します。

虎威では、X の投稿画面に近い操作で本文、画像、動画、投稿予定時刻を設定できます。AI API Key を登録している場合は、キーワードから投稿案を作成することもできます。

## 1. 投稿先の X アカウントを開く

左側メニューの「X アカウント」から、投稿先のアカウントを選択します。

![X投稿先アカウントの選択](/img/torai/x-accounts/x-posts0.jpg)

アカウント詳細画面または投稿一覧画面が開きます。

![Xポスト一覧](/img/torai/x-accounts/x-posts1.jpg)

## 2. 新しいポストを作成する

「新規作成」または投稿作成ボタンをクリックするとポスト作成フォームが開きます。

![Xポスト作成](/img/torai/x-accounts/x-posts2.jpg)

1.  ポストのテキストを入力します。
2.  PC内の画像・動画を選択します。
3.  Gドライブ内の画像・動画を選択します。
4.  絵文字を選択します。
5.  日本語140文字の制限の使用率を表示します。
6.  投稿タイプを選択します。
7.  投稿日時を指定します。

:::caution
X プレミアム以上の有料プランの方は長文投稿が可能ですが、虎威では現状140文字以上は入力できない仕様です。
:::

### ポスト作成

テキストを入力します。

画像・動画はPC内、Gドライブから選択することができます。PC内の画像・動画を選択した場合でも保存はGドライブに保存され、GoogleシートのApps ScriptはGドライブの画像を投稿に利用します。
![X投稿先アカウントの選択](/img/torai/x-accounts/x-posts9.jpg)

選択した画像は表示されますが、動画はサムネイルが表示されずアイコン表示となります。

![投稿作成画像](/img/torai/x-accounts/x-posts12.jpg)

投稿タイプは、

- 通常ポスト
- リプライ
- 引用リポスト

から選択できますが、以下の制限があります。

![X投稿の種類](/img/torai/x-accounts/x-posts3.jpg)

:::info
リプライは、リプライ相手があなたをメンションしている、または、あなたの投稿を引用している場合のみ可能です。それ以外はエラーとなります。
:::

:::info
引用リプライはリンク付き投稿と見なされ$0.20が加算されます。
:::

投稿日時は現在時刻よりも1時間先から指定することができます。
![X投稿日時の設定](/img/torai/x-accounts/x-posts4.jpg)

入力が完了しましたら「保存」ボタンをクリックします。
![X投稿の保存](/img/torai/x-accounts/x-posts13.jpg)

Gドライブとの連携が未設定の場合には、Google認証を求められます。すべてのチェックをONにして「続ける」をクリックします。
![Gアカウントとの連携](/img/torai/x-accounts/x-posts10.jpg)

保存された投稿予定データは、一覧に追加されます。
![X投稿一覧](/img/torai/x-accounts/x-posts14.jpg)

アクティビティ　＞ データ管理 ＞ 投稿予定では、すべての投稿データを確認できます。
![投稿予定データ](/img/torai/x-accounts/x-posts15.jpg)

Googleスプレッドシートとも同期しています。
![投稿シート](/img/torai/x-accounts/x-posts16.jpg)

## 3. 画像または動画を追加する

画像や動画を添付する場合は、メディア追加ボタン（PC内・Gドライブ）からファイルを選択します。

![X投稿先アカウントの選択](/img/torai/x-accounts/x-posts9.jpg)

虎威では投稿には、Google ドライブ上のファイルを使用します。ファイルのアップロードや選択が必要な場合は、画面の案内に従って進めます。

下図は、Gドライブボタンをクリックした場合の表示でGドライブ内の画像・動画データが表示されます。

![Gドライブからの選択](/img/torai/x-accounts/x-posts17.jpg)

:::caution
動画はファイルサイズが大きくなりやすく、Apps Script の実行時間制限に影響します。大きな動画を投稿する場合は、事前にテスト投稿を行ってください。

X側の制限は512MBです。
:::

## 4. 投稿予定時刻を設定する

投稿する日時を設定します。

![X投稿日時の設定](/img/torai/x-accounts/x-posts4.jpg)

設定可能な投稿日時は現在時刻よりも1時間先です。

### 一括で投稿時刻を設定する

一括で投稿時刻を設定することもできます。

- 開始日、終了日
- 開始時刻、終了時刻
- 投稿間隔

を設定することで、選択した投稿データに一度で時刻を設定することができます。

1. 一括投稿時刻セットするポストを選択します。
   ![一括予約投稿設定](/img/torai/x-accounts/x-posts_sche1.jpg)

2. 一括投稿時刻設定ボタンをクリックします。
   ![一括予約投稿設定](/img/torai/x-accounts/x-posts_sche2.jpg)

3. 開始日、終了日、開始時刻、終了時刻、間隔を入力し「作成」ボタンをクリックします。
   ![一括予約投稿設定](/img/torai/x-accounts/x-posts_sche3.jpg)

4. 投稿時刻が設定されます。
   ![一括予約投稿設定](/img/torai/x-accounts/x-posts_sche4.jpg)

## 5. AI で投稿案を作成する

Gemini API Key を登録している場合は、キーワードから投稿案を作成できます。

1. 投稿先のポスト一覧にある「AI」ボタンをクリック

![AIでポスト作成](/img/torai/x-accounts/x-posts_ai1.jpg)

2. キーワード欄に入力し「ポスト案を作成」ボタンをクリックします。
   ![AIでポスト作成](/img/torai/x-accounts/x-posts_ai2.jpg)

3. 作成された案は、すべて編集可能です。投稿に採用するポストにチェックを入れ、「採用したポストを取込」ボタンをクリックします。
   ![AIでポスト作成](/img/torai/x-accounts/x-posts_ai3.jpg)

4. ポスト一覧に表示されます。
   ![AIでポスト作成](/img/torai/x-accounts/x-posts_ai4.jpg)

:::caution
AI が生成した文章は必ず確認してください。誤情報、過度な表現、意図しない表現が含まれる場合があります。
:::

## 投稿前のチェック

- 投稿先アカウントは正しいか
- 本文に誤字や不要なURLがないか
- 画像や動画は正しいか
- 投稿予定時刻は意図した日時か
- スレッドにする投稿ではないか
- API利用量や課金に問題がないか
