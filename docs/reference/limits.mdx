# 制限と注意事項

虎威は複数の外部サービスを利用するため、それぞれの制限を確認してください。

## Google

- Apps Script の実行時間
- 外部アクセス回数
- トリガー数
- Google ドライブ容量

## X

- API 利用量
- 投稿、リプライ、スレッド投稿に関する制限
- メディアアップロードに関する制限
- 従量課金と Usage

## Gemini

- API 利用量
- 課金設定
- 生成結果の確認

## 虎威

- サブスクリプション状態
- 登録可能なアカウント数
- 利用できる機能の範囲
