# 用語集

## API Key

X API でアプリを識別するためのキーです。

## API Key Secret

API Key と組み合わせて使う秘密情報です。

## Access Token

X アカウントの権限で API を実行するためのトークンです。

## Access Token Secret

Access Token と組み合わせて使う秘密情報です。

## Apps Script

Google が提供するスクリプト実行環境です。虎威では Google シート上の投稿処理に使います。

## トリガー

Apps Script を定期的に実行する仕組みです。
