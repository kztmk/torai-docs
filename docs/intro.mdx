# はじめに

虎威は、あなたの Google アカウントで使える Apps Script、スプレッドシート、Google ドライブを連携させて、X へのポスト作成支援と自動投稿を行う Web アプリケーションです。

スプレッドシートの行や列を細かく操作する必要はありません。X に投稿する感覚でポストを作成し、投稿日時や自動投稿の設定を進められます。

![虎威で使うサービスの流れ](/img/intro-google-x-flow.svg)

## 虎威でできること

虎威では、X へ投稿するための作業をひとつの画面から進められます。

- ポストの作成を支援する
- 指定した日時に投稿する
- 自動投稿の設定を行う
- Google ドライブやスプレッドシートと連携して投稿データを扱う
- Discord 通知を使ってスマホで自動投稿の状況を受け取る

普段の投稿に近い操作で使えるため、スプレッドシートを直接編集することに慣れていない方でも始めやすいのが特徴です。

## 格安料金で使える月額制サービス

虎威は、月額制サブスクリプションで提供されています。

利用はいつでも開始・停止できます。停止したあとも、アカウントを削除しない限り、必要になったタイミングで再開できます。

:::info
サブスクリプションを停止すると、虎威の有料機能は利用できなくなります。再開すると、同じアカウントで再び利用できます。
:::

## はじめるために必要なもの

虎威を使い始める前に、次の 2 つを準備してください。

- Google アカウント
- X アカウント

Google アカウントは、Apps Script、スプレッドシート、Google ドライブとの連携に使用します。X アカウントは、実際にポストを投稿するために使用します。

## あると便利なもの

Discord アカウントがあると、自動投稿の結果や通知をスマホで受け取れます。

Discord は無料で利用できます。必須ではありませんが、外出中でも投稿状況を確認したい方にはおすすめです。
