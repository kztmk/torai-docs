# はじめに

虎威は、X のポスト作成、予約投稿、スレッド投稿、AI による投稿案作成を支援する Web アプリケーションです。

旧版マニュアルでは買い切り利用と X API の無料枠を前提にしていましたが、現行版では次の 2 点を先に理解しておく必要があります。

- X API の利用は従量課金を前提に確認する
- 虎威の提供形態はサブスクリプションとして扱う

## このマニュアルの読み方

初めて利用する場合は、次の順で読み進めてください。

1. 利用前の準備
2. 初期設定
3. 虎威の使い方
4. サブスクリプションと利用状況の確認
5. 困ったとき

## 旧版からの主な変更点

### X API の扱い

X の開発者サイトで X アカウントごとに Project / App を作成し、API Key、API Key Secret、Access Token、Access Token Secret を取得します。

投稿は X API の利用量に応じて課金される前提で、X Developer Console の Usage や課金状況を確認してください。

### 虎威の契約

虎威の利用にはサブスクリプション契約が必要です。契約状態、更新、解約、支払い方法は「サブスクリプション」章で説明します。

## 旧版マニュアルから引き継ぐ内容

旧版の章立てから、Google シート / Apps Script の設定、X アカウント管理、ポスト作成、スケジュール、スレッド投稿、自動投稿、メンテナンスの流れを引き継ぎます。

画面や外部サービスの仕様変更がある箇所は、現行版に合わせて順次更新します。
