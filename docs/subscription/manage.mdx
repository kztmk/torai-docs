# サブスクリプション管理

虎威はサブスクリプションとして提供されます。

サブスクリプションの支払いは、

- 銀行振り込み（6ヶ月プランのみ、事務手数料880円）
- クレジットカード（1ヶ月、6ヶ月、1年プラン）

が選択できます。

サブスクリプション契約完了後は、いつでも解約が可能です。ただし期限まで使えますが返金は行いません。

## 契約プランの確認

サブスクリプションが無い場合には、サブスクリプションページに自動で移動します。

サブスクリプション契約がある場合には、プロフール　＞ サブスクリプションで確認いただけます。

ご確認いただける情報は、

- 支払い方法の確認
- 契約更新日の確認

となります。

### 銀行振り込みの場合

銀行振り込みでお申し込みの場合には、

- 銀行振り込みで申し込み
- 振り込み完了
- 振り込み名義を入力し確認依頼
- 確認後、サブスクリプション開始

となり、以下の画面になります。

![銀行振り込みでのサブスクリプション](/img/torai/post/subsc3.jpg)

### クレジットカードの場合

クレジットカードでのサブスクリプションのお申し込みはStripeでの決済となります。

お申し込みのクレジットカードに関する情報はStripeとの直接となり、当方には契約内容しか届きません。

解約は、サブスクリプションページにある「」ボタンをクリックしていただくと、Stripeのページが開きます。

![クレジットカードでのサブスクリプション](/img/torai/post/subsc1.jpg)

## 解約方法

解約はいつでも行っていただけます。解約は次回更新日に課金が行われません。

解約しても次回更新日まではご使用いただけます。返金は行いません。

### クレジットカード

「プロフィール　＞ サブスクリプション」にある「契約情報を変更・確認する」ボタンをクリックします。
![サブスクリプション情報](/img/getting-start/torai/subscription03.jpg)

Stripeのお客様個人ページが表示されます。「Cancel subscription」をクリックします。
![サブスクリプションキャンセル](/img/getting-start/torai/subscription04.jpg)

理由を尋ねられますが、「Continue to cancellation」ボタンをクリックします。
![サブスクリプションキャンセル理由](/img/getting-start/torai/subscription05.jpg)

「Cancel subscription」ボタンをクリックします。
![サブスクリプションキャンセル実行](/img/getting-start/torai/subscription06.jpg)

プランがキャンセルになったことを確認してください。次回更新時の課金は行われません。
![サブスクリプションキャンセル完了](/img/getting-start/torai/subscription07.jpg)

### 銀行振り込み

銀行振り込みでのサブスクリプションは、期限の10日前に更新のご案内をメールで行います。入金を行わなければ、期限をもって自動的に退会となります。

## 契約終了後の利用可否

　「プロフィール ＞ アカウント情報」でアカウントを削除しなければ、いつでも再開できます。

## 領収書や請求履歴の確認

クレジットカードの場合には、Stripeのユーザーページで請求履歴をご確認いただけます。銀行振り込みの場合には、銀行振り込み確認メールをもって領収書とさせてください。

## 注意点

X API の従量課金は、虎威のサブスクリプションとは別に X 側で発生します。
