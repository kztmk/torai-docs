# 紹介報酬制度

虎威には、紹介した方の利用状況に応じて報酬を受け取れる紹介プログラムがあります。

## 紹介リンクを確認する

1. 虎威にサインインします。
2. 左側のメニューから「プロフィール」を開きます。
3. 「紹介プログラム」タブを選択します。
4. 「紹介リンク」の右にある「コピー」ボタンをクリックします。
5. コピーしたリンクを、虎威を紹介したい方へ共有します。

![プロフィールの紹介プログラム画面](/img/subscription/referral-program.png)

:::caution
報酬の対象として正しく記録されるよう、紹介する方には必ずご自身の紹介リンクから登録してもらってください。
:::

## 報酬が確定する条件

紹介された方が、次のいずれかを完了すると報酬が確定します。

- クレジットカードで初回決済を完了する
- 銀行振り込みの承認を完了する

紹介リンクを開いただけの場合や、登録後に決済・承認が完了していない場合は、報酬は確定しません。

## 紹介人数に応じた報酬

紹介プログラム画面では、次の目標と報酬を確認できます。

| 紹介人数 | 報酬 |
| ---: | --- |
| 1人 | 1ヶ月無料 |
| 5人 | 追加3ヶ月無料 |
| 10人 | 追加6ヶ月無料 |
| 30人 | 1年無料 |
| 50人 | 永久50%OFF |
| 100人 | 永久無料 |

「次の報酬」には、次の目標までに必要な紹介人数と現在の達成状況が表示されます。

:::info
紹介人数や報酬の適用状況は「紹介プログラム」画面で最新の内容をご確認ください。
:::

## 紹介状況と報酬を確認する

画面では、紹介プログラムの状況を次の項目で確認できます。

- **紹介登録**: 紹介リンクから登録した人数
- **規約同意完了**: 利用規約への同意を完了した人数
- **サブスク開始**: サブスクリプションを開始した人数
- **獲得月数**: 紹介により獲得した無料期間
- **獲得報酬**: 確定した報酬額
- **付与済み**: すでに付与された報酬額
- **未付与**: まだ付与されていない報酬額
- **消化済み目安**: 使用済み報酬の目安

「利用可能な報酬」には、Stripe customer balance と銀行振込クレジットの合計が表示されます。

表示が最新の状態でない場合は、画面右上の「更新」ボタンをクリックしてください。
