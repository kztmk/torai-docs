# 料金と外部 API 費用

虎威を利用する際の費用は、大きく分けて次の 2 種類です。

## 虎威の利用料

虎威のサブスクリプション料金です。

## 外部サービスの費用

虎威から利用する外部サービス側で発生する費用です。

- X API の従量課金
- Gemini API の利用量に応じた費用
- Google アカウントや Google ドライブの容量に関する費用

## 確認する場所

- 虎威の契約状態: 虎威のサブスクリプション画面
- X API の利用量と課金: X Developer Console
- Gemini API の利用量と課金: Google 側の管理画面
