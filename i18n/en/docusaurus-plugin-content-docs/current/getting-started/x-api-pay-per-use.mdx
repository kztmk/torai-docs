# Check X API Pay-per-use Pricing

The current version assumes that you are billed according to your X API usage.

When you post, reply, create threads, or send DMs through the API, check your usage and billing status in the X Developer Console.

## What to check

- You can sign in to the X Developer Console
- You have a Project / App for each X account that will post
- Your credits or billing settings are configured
- You can view usage on the Usage page
- You have checked the cost per post and your usage limits

## Using multiple accounts

When you manage multiple X accounts with Torai, obtain a complete set of API keys for each X account.

Name and store the credentials so that you can identify which X account each set belongs to.

:::caution
As a security measure, Torai encrypts all information related to your X accounts and stores it in your own Google Sheet.

Torai cannot display these credentials again after they have been saved.
:::
