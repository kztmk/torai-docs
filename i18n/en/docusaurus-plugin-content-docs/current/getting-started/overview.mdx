# Before You Start

Prepare the following before using Torai.

## Requirements

- A Torai subscription (credit card)
- An X account to post from
- A Google Account
- A Gemini API key (if you want to create posts with AI)
- A Discord account (if you want to receive Torai posting results)

## Potential costs

- A monthly or annual Torai subscription
- Pay-per-use X API charges
