# Prepare a Google Account

Torai uses Google Sheets, Google Apps Script, and Google Drive to manage post data, images, and videos.

![Torai automated posting flow](/img/getting-start/torai_flow.jpg)

## What you need

- A Google Account
- Permission to create Google Sheets
- Permission to deploy Apps Script
- Enough Google Drive storage for images and videos

## Important notes

Google Sheets and Apps Script are at the center of the posting process.

If you want to use a separate Google Account for posting, keep a record of which account is associated with each set of X API keys, Google Sheet, and Torai configuration.
