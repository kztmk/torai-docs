# Get a Gemini API Key

To use Torai's AI post-draft feature, get a Gemini API key from Google AI Studio.

Gemini, provided by Google, offers limited free access to AI through an API—an interface that applications can call directly.

## Gemini free tier

You can use Gemini free of charge within the following limits.

Torai uses Flash and Flash-Lite models.

| Model | Requests/minute | Requests/day | Tokens/minute |
| :--- | :--- | :--- | :--- |
| Gemini 2.5 | 15 | 1,500 | 1 million tokens |
| Gemini 3.1 | 30 | 2,000 | — |

For many users, 30 requests per minute and 2,000 requests per day should be sufficient.

:::caution
Gemini API pricing and free-tier limits may change. Check Google's latest information before using the API.
:::

## Get the API key

### 1. Open Google AI Studio

Open [Google AI Studio](https://aistudio.google.com) and click **Get started**. If you are not signed in to Google, you will need to sign in.

![Google AI Studio](/img/getting-start/gemini_access.jpg)

### 2. Click the key icon

Click the key icon in the lower-left corner.

![Google AI Studio key button](/img/getting-start/gemini_try_key.jpg)

### 3. Confirm the key

Your key is displayed. Confirm that it belongs to the free tier.

![Gemini API key](/img/getting-start/gemini_get_key.jpg)

### 4. Copy the key

Click the key to display its details, then copy it.

![Copy Gemini API key](/img/getting-start/gemini_copy_key.jpg)

You have now obtained your Gemini API key.
