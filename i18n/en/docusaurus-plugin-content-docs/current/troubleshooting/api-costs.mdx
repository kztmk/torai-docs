# If You Are Concerned About API Usage or Charges

External API costs may be charged separately from your Torai subscription.

## X API

Check Usage and billing settings in the X Developer Console.

Before scheduling a large number of posts or threads, check the number of planned posts and your current usage.

## Gemini API

Check usage and billing settings in the relevant Google management console.

Before generating a large number of AI post drafts, check your usage limit and billing settings.
