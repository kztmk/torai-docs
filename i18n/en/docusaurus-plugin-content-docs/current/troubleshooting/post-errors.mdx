# Posts Are Not Being Published

If publishing to X fails, check the following.

## Checklist

- All four X API credentials are correct
- App permissions are set to **Read and write** or higher
- You regenerated the Access Token after changing permissions
- There are no issues with X API billing or Usage
- The Google Apps Script web app URL is correct
- The trigger is running
- The **Errors** sheet in Google Sheets contains a relevant entry
