# Glossary

## API Key

A key that identifies your application to the X API.

## API Key Secret

Secret information used together with the API Key.

## Access Token

A token used to call the API with the permissions of an X account.

## Access Token Secret

Secret information used together with the Access Token.

## Apps Script

Google's script execution environment. Torai uses it to process posts stored in Google Sheets.

## Trigger

A mechanism that runs Apps Script on a schedule.
