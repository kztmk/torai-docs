# Limits and Important Notes

Torai uses multiple external services. Check the limits that apply to each service.

## Google

- Apps Script execution time
- External request quota
- Number of triggers
- Google Drive storage

## X

- API usage
- Limits for posts, replies, and threads
- Media upload limits
- Pay-per-use charges and Usage

## Gemini

- API usage
- Billing settings
- Review of generated output

## Torai

- Subscription status
- Number of accounts you can register
- Available features
