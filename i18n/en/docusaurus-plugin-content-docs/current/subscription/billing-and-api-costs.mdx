# Fees and External API Costs

There are two main types of costs when using Torai.

## Torai subscription fee

This is the subscription fee for using Torai.

## External service costs

These costs are charged by external services used through Torai.

- Pay-per-use X API charges
- Gemini API usage charges
- Fees related to your Google Account or Google Drive storage

## Where to check

- Torai subscription status: the Subscription page in Torai
- X API usage and charges: X Developer Console
- Gemini API usage and charges: the relevant Google management console
