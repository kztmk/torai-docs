# 虎威 V2　サービス開始

## 虎威 V2 とは？

　「虎威 V2」は、XのAPI使用が従量課金に変更になったことに追従してリニューアルしたサービスです。

### 月額のサブスクリプション開始

　一番の変化は、月額のサブスクリプションを取り入れ低額で提供できるようになったことです。

### 紹介報酬サービスも

　また、紹介報酬サービスも追加しています。

　あなたの紹介コードでサブスクリプションにお申し込みいただくと、虎威の使用料金が無料になったりします。

　是非、様々なメディアで紹介をお願いします。

### セキュリティの大幅アップ

　個人コードでGoogleシートと紐付けすることで完全に他人を排除しました。あなたのGoogleシートには、虎威へログインした「あなた」以外は読み込み・書き込みができません。

### 充実のサポートは従来通り

　365日24時間以内の返信を続けています。ご不明な点、不具合などご遠慮なくお知らせください。
